# Where's Hosp

病院検索に特化したアプリケーションです。<br>
半径1km内の医療施設を探索でき、評価情報も記載されています。<br>
また口コミ機能も搭載しているため、実際に病院に訪れた人の感想をリアルタイムで見ることができます。<br>

## URL

https://wherehosp.com
